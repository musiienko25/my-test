# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md) uses [Babel](https://babeljs.io/) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh


1. At the bottom of the home page, copy the products in "Top Brands" into the "Best Sellers" field.
2. Add the "About us" in the navigation bar. When it is clicked, the content of "About us" should be displayed.
3. Frontend cannot communicate with Backend. There is one important line of code is missing in the backend project. Find that part and add one line of code. Hint: All the URLs start with localhost:5000/api/.
